# Agentic AI Workflows Notes

Agentic workflows combine planning, tool use, and iterative refinement.

Typical steps:
1. Interpret the user request
2. Decide which tools to use
3. Gather evidence from web and local sources
4. Synthesize findings into a structured answer

Best practices:
- Keep tool outputs small and relevant
- Use a structured final schema
- Log references and sources
- Separate evidence gathering from final writing
